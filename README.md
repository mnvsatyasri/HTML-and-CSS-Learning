# HTML-and-CSS-Learning

### PROJECT – Website (HTML)
#### Algorithm
1. Start by creating an HTML file named website.html.

2. Add a `<button>` element with the text Hello.

3. Insert a `<p>` tag containing a sample paragraph.

4. Include an `<a>` tag linking to YouTube, with target="_blank" to open in a new tab.

5. Save the file and open it in a browser to view the output.

6. End — basic HTML practice project complete.

### PROJECT - Controls (HTML + CSS)

#### Algorithm
1. Create an HTML file named `buttons.html`.
2. Link a CSS file named `buttons.css` inside the `<head>` section.
3. Add three `<button>` elements with different classes:
   - `subscribe-button`
   - `join-button`
   - `tweet-button`
4. In `buttons.css`, style each button with properties like:
   - Background color
   - Text color
   - Border and border-radius
   - Height and width
   - Cursor type
   - Margins
5. Save both files and open `buttons.html` in a browser.
6. End — styled buttons displayed successfully.

#### Technologies Used
- HTML for structure
- CSS for styling
